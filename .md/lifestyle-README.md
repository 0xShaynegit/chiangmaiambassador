# Lifestyle Section - Template Standard

**All pages in this section use template.html structure. No custom markup. No nav changes.**

## Creating a New Page in /lifestyle/

1. Copy template.html from project root
2. Save as your-page-name.html here
3. Update **only content** (headings, paragraphs, sidebar)
4. Use `../` prefix for all paths (css/, fonts/, images/, js/)
5. Done
